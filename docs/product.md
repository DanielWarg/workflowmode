# Product (kort)

## North Star

“Jag beskriver hur det ska funka → appen bygger ett komplett workflow (grafiskt) → jag finslipar med prompt.”

## V1-fokus (Wedge)

**Meeting Mode**: Klistra in mötesanteckningar → få ett tydligt beslut- och action-flöde på < 60 sekunder.

## Kärnprinciper

- **Workflow mode är default**
- **Canvas = rendering av WorkflowSpec**
- **Preview → Apply** (trygghet)
- **Determinism** (stabil output och idempotent apply)
- **AI är optional** (editorn ska fungera även utan AI)

## UX-principer (V1)

- **Navigator: list ↔ flow**: “Åtgärder/Öppna frågor/Uppföljning” fungerar som navigationslager (klick → zoom/highlight nod).
- **Fit-to-view**: Vid generering ska hela workflowet synas direkt (ingen manuell zoom för att “hitta flödet”).

## Non-goals (V1)

- Full Miro-paritet
- Video calls
- “250 integrationer”
