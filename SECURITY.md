# Security Policy

## Reporting a Vulnerability

Om du hittar en sårbarhet, rapportera den **privat**.

- Skicka en kort beskrivning, reproduktionssteg och eventuell PoC
- Inkludera påverkan (vad kan en angripare göra?) och föreslagen fix om du har en

**Kontakt:**
- Öppna inte en publik issue för säkerhetsproblem.
- Kontakta repo-ägaren via GitHub-profilen för `@DanielWarg` (se `CODEOWNERS`).

## Supported Versions

Det här är ett tidigt projekt. Vi supportar senaste versionen på `main`.
